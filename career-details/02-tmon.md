# 티몬 (TMON) — 상세 경력기술서

> 보안팀 / 대리 | 2020.07 ~ 2023.02 (2년 8개월) | 이커머스

---

## 핵심 역할

이커머스 기업의 보안 담당자로서 보안솔루션 운영, ISMS-P/ISO27001 인증심사 대응,
전사 재택근무 인프라 구축, 보안 관제 자동화 등 수행.

---

## 주요 프로젝트

### 1. O365 보안기능 점검 및 MS Cloud 보안 구축
- **기간**: 2022.03 ~ 2023
- MFA, DLP, AIP 등 보안기능 점검 및 적용

### 2. 스마트워크(전사 재택근무) 환경 구축
- **기간**: 2022.02 ~ 2022.08
- SSLVPN All traffic 설정, 엔드포인트/트래픽 통제 담당
- 전사 재택근무 보안 환경 구축 완료

### 3. 서버실 이전 준비
- **기간**: 2022.02 ~ 2022.06
- 공인 IP 변경 영향도 파악, Radius → LDAP 방식 AD 직접 연결 준비

### 4. SSL VA / Proxy 장비 노후화 교체 PoC
- **기간**: 2022.02 ~ 2022.03
- 암복호화 장비 교체 PoC, 사내 인증서 등록 및 전사 적용

### 5. 보안기술팀 업무공유 페이지 구축
- **기간**: 2022.01 ~
- 팀 내 업무공유, 매뉴얼 최신화, 형상관리 목적 제안 및 운영

### 6. 사내 서버실 화재 대응
- **기간**: 2022년
- 네트워크 우회 방식 제안, SSLVPN 장비를 IDC로 이전
- 대외서비스 장애 없이 임직원 업무 환경 신속 복구

### 7. ISMS-P / ISO27001 / ISO27701 인증심사 대응
- **기간**: 2021 ~ 2022 (매년)
- FW, VPN, DBsafer, DRM, DLP, NAC, SEP, PMS 담당 인터뷰
- 보안 컨설팅 없이 자체 대비하여 인증 획득

### 8. WIPS PoC/BMT 총괄
- **기간**: 2021.05
- 본사 WIPS 교체를 위한 PoC/BMT 총괄 진행

### 9. 보안 관제 자동화 시스템 구축
- **기간**: 2021.04 ~
- 연간 약 1.5억원 보안 관제 서비스 비용 내재화
- ELK 수집 방화벽 로그 기반 공격 의심 IP 자동 차단
- Fortigate API 활용 Python 자동 차단 스크립트 개발
- 외부 관제 서비스 없이 ISMS-P 인증 획득, 연 1.5억 비용 절감

### 10. 시스템 취약점 점검 및 이행점검
- **기간**: 2021.03 ~ 2021.09
- 서버, WAS, DB, 네트워크, 정보보호시스템 대상 점검

### 11. 콜센터 IPSec VPN 장비 교체
- **기간**: 2021.03
- 본사, 지사(삼성, 대구) VPN 장비 노후화 교체
- 고객 콜센터 서비스 무중단 교체 완료

### 12. 긴급 전사 재택근무 시스템 구축 (SSLVPN)
- **기간**: 2020.12 ~ 2021.01
- 코로나19로 긴급 전사 재택근무 필요
- Pulse Secure VPN 장비 도입, 전사 재택근무 VPN 보안 정책 수립
- SSLVPN 장애율 30% → 5%로 감소

## 사용 기술

| 분류 | 기술 |
|------|------|
| 방화벽 | Fortigate |
| VPN | Pulse Secure (SSLVPN), IPSec VPN |
| 보안솔루션 | DRM, DLP, NAC, SEP(백신), PMS, DBsafer, WIPS, IPS |
| 로그분석 | ELK Stack (Elasticsearch, Logstash, Kibana) |
| 개발 | Python (자동 차단 스크립트) |
| 클라우드 | MS O365, Azure AD (MFA, DLP, AIP) |
| 인증 | ISMS-P, ISO27001, ISO27701 |
